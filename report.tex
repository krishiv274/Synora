\section{Problem Statement}

The rapid adoption of Electric Vehicles (EVs) has outpaced charging infrastructure planning, resulting in congestion in high-demand zones and underutilization elsewhere. This project builds a predictive analytics system to forecast:

\begin{itemize}
    \item Occupancy (\%)
    \item Charging Volume (kWh)
\end{itemize}

The goal is to support intelligent infrastructure expansion and congestion mitigation.

% ------------------------------------------------
\section{Data Description}

Dataset: \textbf{UrbanEV (Shenzhen, China)}

\begin{itemize}
\item 275 Travel Analysis Zones
\item 1,362 charging stations
\item 17,532 charging piles
\item Timeframe: Sept 2022 – Feb 2023 (Hourly)
\end{itemize}

% ------------------------------------------------
\section{Methodology}

We implemented an ensemble learning framework using:

\begin{itemize}
\item Random Forest
\item XGBoost
\item LightGBM
\end{itemize}

\subsection{Feature Engineering}

\textbf{Lag Features:}
1h, 3h, 6h, 12h, 24h, 168h

\textbf{Cyclical Encoding (4 features)}

To preserve the circular nature of time:

\begin{equation}
\text{hour\_sin} = \sin\left(\frac{2\pi \cdot hour}{24}\right)
\end{equation}

\begin{equation}
\text{hour\_cos} = \cos\left(\frac{2\pi \cdot hour}{24}\right)
\end{equation}

\begin{equation}
\text{dow\_sin} = \sin\left(\frac{2\pi \cdot day\_of\_week}{7}\right)
\end{equation}

\begin{equation}
\text{dow\_cos} = \cos\left(\frac{2\pi \cdot day\_of\_week}{7}\right)
\end{equation}

\section{Evaluation Metrics}

Four metrics were used:

\subsection*{Mean Absolute Error (MAE)}
\[
MAE = \frac{1}{n}\sum_{i=1}^{n} |y_i - \hat{y}_i|
\]

\subsection*{Root Mean Square Error (RMSE)}
\[
RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}
\]

\subsection*{$R^2$ Score}
\[
R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}
\]

\subsection*{Mean Absolute Percentage Error (MAPE)}
\[
MAPE = \frac{100}{n} \sum_{i=1}^{n} \left| \frac{y_i - \hat{y}_i}{y_i} \right|
\]
